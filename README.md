# 中国民用航空局(CAAC) 国际定期客运航班预先飞行计划爬虫

## 介绍
鉴于当前疫情影响，国际航班频繁熔断,机票价格不透明，票代猖獗。以及航空公司袖手旁观，甚至出售无效票以获得无息贷款。该项目用于查询民航局批准的国际客运航班，让需要回国的人更方便的找到可用的航班，让虚假票无处遁行。

## 运行
```Python
python3 start.py
```
